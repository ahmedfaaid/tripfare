# Data fields

## User

- first name
- last name
- email
- username
- profile picture
- date of birth
- location (city, state, country)
- occupation
- socials

## Post

- title
- region (continent)
- country
- city
- details (probably collected as markdown)
- media (images or video)
- length of stay (days, weeks, months)
- date travelled
- size of group
- total budget (places can be added for items with and connected to affiliate links)
  - accomodation
  - food & drinks
  - activities
  - transportation
